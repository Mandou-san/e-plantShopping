# e-plantShopping 🌿

Welcome to **e-plantShopping**! a React + Redux web application that allows to browse a variety of plants and manage a shopping cart in real time.
Project for IBM specialization course. Made by Armando Vidaurri.

## ✨ Features
This project simulates an online plant store where users can: 
- Browse plants by category
- Add plants to a shopping cart
- Increase and decrese item quantities
- Remove items from cart
- View total cost and item subtotals
- See the total number of items in the cart icon
- Continue shopping from the cart page

## 🛠️ Tech Stack
- React
- Redux toolkit
- React-Redux
- Javascript (ES6+)
- CSS
